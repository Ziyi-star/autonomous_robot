\doxysubsection{Detailed Description}
Holds ADC (Analog-\/to-\/\+Digital Converter) values from multiple channels. 

This structure is designed to store the digital values converted from analog signals on multiple ADC channels. It can be used for easy access and manipulation of these values within the application.
