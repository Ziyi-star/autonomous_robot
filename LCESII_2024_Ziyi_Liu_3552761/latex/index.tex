 \hypertarget{index_autotoc_md1}{}\doxysection{Project Overview}\label{index_autotoc_md1}
This repository contains all the necessary software for controlling an autonomous line-\/following robot designed as part of the Embedded Systems Laboratory at Universit√§t Kassel under the supervision of Herr Benjamin Herwig. The robot is programmed to navigate a line track for three rounds and perform various tasks as dictated by the project requirements.\hypertarget{index_autotoc_md2}{}\doxysubsection{Features}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item Navigates a predefined line track for three consecutive rounds.
\item Uses line-\/following sensors to stay on track.
\item Communicates its status and actions via USART.
\item Capable of pausing, resuming navigation and Rotation based on commands.
\item Implements a watchdog timer for automatic reset after completing its tasks.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{Getting Started}\label{index_autotoc_md3}
\hypertarget{index_autotoc_md4}{}\doxysubsection{Prerequisites}\label{index_autotoc_md4}

\begin{DoxyItemize}
\item AVR-\/\+GCC compiler for building the project.
\item AVRDUDE or a similar tool for flashing the firmware to the microcontroller.
\item An AVR microcontroller ATmega328/P, set up with necessary peripherals (motors, sensors, etc.).
\end{DoxyItemize}\hypertarget{index_autotoc_md5}{}\doxysubsection{Installation}\label{index_autotoc_md5}

Configure your hardware setup and microcontroller settings in {\ttfamily \mbox{\hyperlink{config_8h_source}{config.\+h}}} as required to match your specific hardware.\hypertarget{index_autotoc_md7}{}\doxysection{Usage}\label{index_autotoc_md7}
After the firmware is successfully flashed\+:
\begin{DoxyItemize}
\item Place the robot at a designated start point on the track.
\item Power on the robot.
\item Connect to the robot via Bluetooth and open a serial connection using Cutecom or a similar program.
\item Send commands to control the robot as specified\+:
\begin{DoxyItemize}
\item {\bfseries{S\+:}} Start navigation.
\item {\bfseries{P\+:}} Pause/resume navigation.
\item {\bfseries{T\+:}} Test rotation functionality.
\end{DoxyItemize}
