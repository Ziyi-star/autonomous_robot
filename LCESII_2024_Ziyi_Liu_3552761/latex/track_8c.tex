Motor Control Functions for Robotic Navigation.

This file contains functions to control the movement of a robot through its motor-\/driven wheels. Functions in this file allow for precise movement control, including sharp turns, slight adjustments, and straightforward motion. The control logic includes speed adjustment, direction control, and stopping mechanisms, integrating directly with the robot\textquotesingle{}s hardware interface.
