\doxysubsection{Detailed Description}
LED Control for Robotic Shift Register Interface. 

\begin{DoxyAuthor}{Author}
Ziyi Liu 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
04.\+09.\+2024
\end{DoxyDate}
This file includes functions that interface with the robot\textquotesingle{}s shift register to control LEDs based on specific patterns and timing. Functions provide mechanisms to clock the hardware, update LED states both in memory and on the actual hardware, clear settings, and execute LED sequences to represent various states or signals visually.
