Detailed Description

Motor control definitions and functions for robotic movements.

Author
Ziyi Liu
Date
04.09.2024

This header file contains macros and functions for controlling the directions and movements of a robot's motors. It provides a high-level interface for common maneuvers such as turning and stopping.
