This file contains the main function which serves as the central control hub for an autonomous robot. It integrates sensor data processing, state management, and command execution to handle different operational modes such as navigation, obstacle detection, and system diagnostics. The robot's behavior is influenced by ADC values to dynamically adjust its course and by UART messages to switch between different operational states.
